----------------------README----------------------
Para compilar o codigo execute o seguinte comando:
 
$ make
 
Em seguida, para executar o programa:
 
$ bin/pmt padrao texto.txt
 
O programa segue o seguinte padrao:
pmt [-e error] [-p patternsfile] [pattern] textfile [textfile...]
-h, --help: Apresenta as instrucoes para a utilizacao basica da ferramenta.
 
autores: bap2, phtg, rlfs
