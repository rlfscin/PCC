----------------------README----------------------
Para compilar o codigo execute o seguinte comando:
 
$ make

Para indexar e comprimir um arquivo, execute:

$ bin/ipmt index file.txt [--compression=lz78] [--compression=lzw]

Para efetuar uma busca, execute:

$ bin/ipmt search file.txt.idx [-p --pattern padroes.txt] padrao [-c]
